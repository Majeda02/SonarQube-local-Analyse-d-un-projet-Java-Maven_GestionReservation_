## "# SonarQube-local-Analyse-d-un-projet-Java-Maven_GestionReservation_" 
### Objectif de l’activité
* Mettre en place SonarQube en local (Docker),
* créer un projet,
* générer un token,
* lancer l’analyse d’un projet Java Maven,
* puis interpréter les résultats (Quality Gate, bugs, odeurs de code, vulnérabilités, couverture…).

### Prérequis
* Docker Desktop (ou Docker Engine)
* Navigateur Web
* JDK installé (selon ton projet)
* Maven (ou Maven Wrapper mvnw)
* Un projet Java Maven (présence de pom.xml) **projet gestion-reservation**
